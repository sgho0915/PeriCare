# 🛡️ PeriCare: Patient-Centric IoT Solution

> **"수기 기록의 고통을 훔치고(Note Thief), 무결한 안전을 선물합니다."**

재택 복막투석 환자의 감염 리스크와 수기 기록의 번거로움을 해결하기 위한 **저비용 고효율 IoT 지능형 의료 안전망**입니다. 

---

## 💡 프로젝트 배경 (Background)

복막투석 환자는 병원 대신 가정에서 일상을 유지할 수 있지만, 매일 4회(약 6시간 간격) 엄격한 무균 처치와 30분이 넘는 투약 과정을 평생 스스로 수행해야 합니다. 

* 
**치명적 감염 리스크**: 미세한 외풍이나 먼지 유입으로 인해 환자의 약 50%가 3년 내 심각한 복막염 합병증을 경험합니다. 


* 
**기록의 고통과 불확실성**: 고령 환자일수록 투입/배액량을 수기로 기록하는 과정에서 오기가 빈번하며, 이는 주치의의 오진으로 직결될 수 있습니다. 



**PeriCare**는 환자가 기존에 보유한 링거대에 저비용 IoT 모듈을 부착하는 것만으로 이러한 고통을 해결하는 **'경제적 예방 백신'**입니다. 

---

## ✨ 핵심 기술 (Key Engineering Points)

1. 지능형 환경 스캔 (Infection Control) 

* **풍속/미세먼지 실시간 감시**: 투석 직전 주변 공기 흐름을 스캔하여 창문 개방이나 에어컨 바람 등 감염 요인을 사전 차단합니다.
* 
**데이터 정제**: **이동 평균 필터(Moving Average Filter)**와 **온도 보상 알고리즘**을 적용하여 센서 노이즈를 제거한 신뢰도 높은 데이터를 제공합니다. 



2. 정밀 자동 측정 및 제수량 산출 

* **Dual-Loadcell 시스템**: 상단 로드셀(주입량)과 하단 4점식 로드셀 발판(배액량/체중)을 통해 데이터를 자동 수집합니다.
* 
**독립적 제수량 계산**: 스마트폰 연결이 끊겨도 ESP32가 자체적으로 $Net\ UF = 배액량 - 주입량$ 공식을 수행하여 데이터 누락을 방지합니다. 



3. 무결성 보장을 위한 페일세이프 (Fail-Safe) 

* 
**하드웨어 이중화**: 정전 시 즉각 개입하는 비상용 Li-Po 배터리를 탑재하였습니다. 


* 
**소프트웨어 무결성**: 통신 장애 시 '수기 입력 모드'로 자동 전환되며, 데이터 수정 시 원본 값과 사유를 남기는 **Audit Log** 시스템을 통해 의료 데이터의 신뢰성을 입증합니다. 



4. 하이브리드 UX 전략 

* 
**노이즈 회피**: 혈압계 모터 등 전력 노이즈 간섭이 큰 데이터는 앱 UI를 통한 수기 입력을 유도하여 시스템 정밀도를 극대화했습니다.

🏗️ 시스템 아키텍처 (System Architecture) 

### 1. Hardware (Retrofit Kit)

* 
**상단**: 주입량 로드셀, 풍속/미세먼지 센서 허브 


* 
**중단**: ESP32 메인 컨트롤러, 비상 배터리, 페어링용 QR 코드 


* 
**하단**: 200kg 급 고정밀 4점 로드셀 체중/배액 발판 



2. Software & Cloud 

* 
**App (Flutter)**: BLE 기반 Wi-Fi 프로비저닝, 실시간 데이터 시각화 리스너 구현 


* 
**Cloud (Firebase)**: 중간 서버를 생략한 **Serverless 아키텍처**로 지연 없는 데이터 스트림 환경 구축
